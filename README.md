# Tower-Defence

Computer Game Technologies (Study Subject)

Instruction to run project:

1. Clone project.
   ```javascript
   git clone https://github.com/Phabes/ Tower-Defence.git
   ```
1. Open "backend" folder in command line.
1. Run:
   ```javascript
   npm install
   ```
1. Run:
   ```javascript
   nodemon server
   ```
1. Open "frontend" folder in second command line.
1. Run:
   ```javascript
   npm install
   ```
1. Run:
   ```javascript
   npm run dev
   ```
